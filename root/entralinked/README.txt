NOTE: Java 17 or newer is required to run Entralinked.

To begin, run "entralinked.jar" by double clicking it.
If this doesn't work for whatever reason, use "start.bat" instead (Windows users only.)

If you get error 20100 when using Game Sync, a potential fix would be to run the program as administrator.
To do this, close Entralinked and start it again by right clicking "start.bat" and selecting the "Run as administrator" option.

Should you have any other issues, please refer to the troubleshooting guide:
https://github.com/kuroppoi/entralinked/wiki/Troubleshooting
